# Helia Chat Application

## Overview
Helia is a modern chat application designed for seamless communication and collaboration among users. It offers a clean interface, efficient messaging, and various features to enhance user experience.

## Features
- **Real-time Messaging**: Engage in instant conversations with your friends or colleagues.
- **Group Chats**: Create group chats to discuss topics with multiple participants.
- **File Sharing**: Share documents, images, and other files effortlessly.
- **User-Friendly Interface**: The app has an intuitive design to make navigation simple.
- **Cross-Platform Support**: Accessible on web, desktop, and mobile.

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/K-z89/Helia.git
   cd Helia
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the application**:
   ```bash
   npm start
   ```

## Usage
- After installation, open the Helia app in your preferred web browser or desktop application.
- Sign up to create an account or log in if you already have one.
- Start a new chat or join existing groups to communicate with others.
